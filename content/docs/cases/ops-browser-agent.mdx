## 建设边界

- 本地排障：端口、进程、容器、数据库、网络、环境变量、日志。
- 远程节点：SSH、部署、健康检查、回滚、告警、事故复盘。
- 浏览器 Agent：DOM、截图、点击、输入、等待、下载、登录态。
- VNC 场景：平台验证、人工接管、远程可视化证据、操作审计。
- 产品化落地：API、队列、任务状态、trace、用户反馈、人工确认。
