案例页不追求堆功能，而是看任务边界、工具选择、状态流转、失败处理和人工接管如何落到系统设计里。

社区开源智能体项目会单独放在 [开源智能体](/docs/open-source-agents) 分组。案例页保留更通用的场景拆解和源码深读。

## 拆解模板

读任何 Agent 案例时，可以固定看五个问题：

1. 任务边界是什么。
2. 哪些步骤必须使用工具。
3. 状态如何保存。
4. 失败如何回放。
5. 什么情况下需要人工接管。
