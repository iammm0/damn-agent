案例页不追求堆功能，而是看任务边界、工具选择、状态流转、失败处理和人工接管如何落到系统设计里。

## 阅读入口

| 案例 | 关注点 |
| --- | --- |
| [Claude Code 源码解析](/docs/cases/claude-code-source-analysis) | 查询引擎、工具系统、Bash 安全、Hook、MCP 和多智能体协作。 |
| [研究型 Agent](/docs/cases/research-agent) | 问题澄清、资料检索、证据整理到结论输出的完整链路。 |

## 拆解模板

读任何 Agent 案例时，可以固定看五个问题：

1. 任务边界是什么。
2. 哪些步骤必须使用工具。
3. 状态如何保存。
4. 失败如何回放。
5. 什么情况下需要人工接管。
