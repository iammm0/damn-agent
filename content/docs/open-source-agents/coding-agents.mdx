这个页面用于承载开源 Coding Agent 项目拆解。它和 `cases/coding-agent` 的关系是：案例页讲通用工作流，本页讲具体社区项目如何实现这些工作流。

## 建设边界

- 仓库理解：文件树、符号、搜索、上下文压缩、依赖识别。
- 编辑机制：patch、批量修改、格式化、冲突处理、用户改动保护。
- 验证机制：lint、typecheck、unit test、build、browser check。
- Git 协作：diff、暂存、commit、PR、review 修复。
- 工具权限：Shell、文件系统、网络、浏览器、包管理器。
- 产品形态：CLI、IDE 插件、Web UI、远程 sandbox。

## 需要回答的问题

- 它如何选择要读的文件。
- 它如何决定修改范围。
- 它如何保护用户已有改动。
- 它如何验证修改是正确的。
- 它如何把结果交给人 review。
