这个页面用于承载浏览器、桌面和 VNC 操作相关的开源智能体项目。重点是页面状态、动作执行、截图证据、等待策略和人工接管。

## 建设边界

- 浏览器控制：DOM、截图、点击、输入、等待、下载、登录态。
- 计算机使用：窗口、坐标、键盘、文件、系统权限。
- 远程操作：VNC、节点浏览器、可视化证据、操作审计。
- 任务恢复：失败重试、页面变化、登录失效、验证码和人工接管。
- 安全边界：账号、cookie、敏感数据、外部提交动作。
