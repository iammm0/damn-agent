本站维护者名单基于远端 GitHub 仓库 collaborators 接口同步，用于标识内容维护和工程协作责任。

<MaintainersSync />

## 维护规则

- 新增或移除远端仓库协作者后，本页会在下一次构建或缓存刷新时自动同步。
- 本页同步展示 GitHub 头像、公开昵称、简介、ID 与仓库权限。
- 本地开发与部署需配置 `GITHUB_TOKEN`，否则无法读取 collaborators 接口。

## 参考来源

- [GitHub REST API: List repository collaborators](https://docs.github.com/en/rest/collaborators/collaborators?apiVersion=2022-11-28#list-repository-collaborators)：维护者列表的数据来源。
- [GitHub REST API: Authentication](https://docs.github.com/en/rest/authentication/authenticating-to-the-rest-api?apiVersion=2022-11-28)：本地构建和部署时 `GITHUB_TOKEN` 的认证依据。
