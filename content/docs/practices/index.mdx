## 工程判断

上线前至少要确认：

1. 工具调用是否有输入 schema、权限和失败语义。
2. 长任务是否能恢复、回放和审计。
3. 输出质量是否有可重复的评测方式。
4. 高风险动作是否需要人工确认。
5. 外部知识是否有来源、更新时间和引用证据。
