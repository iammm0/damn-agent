这个分组用于先确认网站文档目录是否能承载第一期学习内容，再进入排期和分工。

## 当前用途

| 页面 | 用途 |
| --- | --- |
| [第一期目录评估](/docs/roadmap/phase-1-directory) | 对照第一期 Agent 学习板块，检查现有目录覆盖情况和缺口。 |

## 维护原则

- 先确认目录和内容边界，再拆协作者任务。
- 排期前只固定主题承载位置，不提前写死负责人。
- 每个主题页都要能落到定义、流程、例子、风险、检查清单和参考资料。
- 如果某个主题更适合案例页或资源页，不强行塞进概念页。
