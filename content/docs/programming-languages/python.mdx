Python 是本站第二优先级语言。它不是 Agent 产品化唯一选择，但在模型实验、RAG、数据处理、评测、notebook、LangGraph、LlamaIndex、AutoGen、CrewAI 和传统 ML 生态中仍然非常重要。

## 需要掌握的基础

- 类型和数据结构：dataclass、TypedDict、Pydantic、Protocol、typing。
- 异步模型：asyncio、task、timeout、cancel、streaming。
- 包管理：uv、pip、poetry、virtualenv、requirements、pyproject。
- 数据处理：JSON、CSV、Parquet、PDF、HTML、文本切分。
- 服务化：FastAPI、HTTP client、background task、worker。
- 测试：pytest、fixture、mock、golden task、评测脚本。
- 工程化：ruff、mypy、pre-commit、Docker。

## Agent 工程重点

- RAG pipeline：loader、chunk、embedding、rerank、citation。
- 评测系统：数据集、rubric、LLM-as-Judge、回归脚本。
- 研究 Agent：搜索、资料读取、引用管理、报告生成。
- Python 框架：LangGraph、AutoGen、CrewAI、LlamaIndex、Haystack。
- 与 TypeScript 产品层的边界：API、队列、artifact、数据库。
