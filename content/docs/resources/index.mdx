这个资源索引优先收录能帮助开发者形成工程判断的资料类型，而不是简单堆链接。

## 推荐资源类型

| 类型 | 用途 |
| --- | --- |
| 论文 | 理解方法来源和限制 |
| 框架文档 | 查询 API、部署和最佳实践 |
| 案例仓库 | 学习真实目录结构和测试方式 |
| 评测资料 | 构建回归集和质量指标 |
| 运维资料 | 设计日志、追踪、成本和权限 |

## 维护原则

- 记录资料适合哪个学习阶段。
- 标注是否需要付费、登录或特定环境。
- 对高风险领域保留更新时间。
- 避免只收藏标题党文章。
